20:42 - 20:52

Prix des tokens 🦞

Le meilleur modèle n’est pas toujours le meilleur choix business.

  • Input tokens : ce qu’on envoie au modèle.
  • Output tokens : ce qu’il génère, souvent plus cher.
  • Les agents coûtent vite cher car ils font plusieurs appels.
“À partir de quel coût par tâche l’IA devient trop chère ?”
Option si temps

IA locale 🦞

Le local n’a pas besoin d’être le meilleur. Il doit être suffisant, privé et contrôlable.

  • Documents sensibles : PME, écoles, avocats, fiduciaires.
  • Réduction des coûts API.
  • Autonomie face aux plateformes cloud.
  • Limites : hardware, vitesse, UX.
“Quel usage feriez-vous tourner en local plutôt que dans le cloud ?”
